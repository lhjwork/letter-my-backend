## 📊 성과 측정 지표

### 1. 기술적 성과

- [ ] **페이지 성능**
  - [ ] LCP (Largest Contentful Paint) < 2.5초
  - [ ] FID (First Input Delay) < 100ms
  - [ ] CLS (Cumulative Layout Shift) < 0.1

- [ ] **API 성능**
  - [ ] 캐러셀 API 응답 시간 < 500ms
  - [ ] 이미지 업로드 성공률 > 99%
  - [ ] 에러율 < 1%

### 2. 비즈니스 성과

- [ ] **광고 효과**
  - [ ] 캐러셀 CTR vs 일반 광고 CTR 비교

  - [ ] 광고 수익 증가율

- [ ] **운영 효율성**
